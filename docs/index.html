TikTok Uploader

A desktop tool for uploading MP4 videos to TikTok using the TikTok Content Posting API.

Features

  • OAuth 2.0 authentication with TikTok
  • Inbox upload — send a video to your TikTok inbox to review before publishing
  • Direct post — publish a video immediately with configurable metadata
  • Automatic token refresh
  • Simple GUI built with Tkinter
