Short Video Uploader

A desktop application that lets you upload MP4 videos directly to TikTok — quickly, securely, and with full control over your content.

About This App

Short Video Uploader is a desktop tool designed for content creators and developers who want a streamlined way to publish short-form video content to TikTok. Instead of using the TikTok mobile app, users can upload MP4 files directly from their computer through TikTok's official Content Posting API.

The application uses TikTok's official OAuth 2.0 authentication flow to securely connect your TikTok account. Your credentials are never stored by this application — authentication is handled entirely through TikTok's own login system.

This tool is intended for individual creators and developers testing TikTok API integrations. All data processing is performed locally on the user's machine.

Features

🔐

Secure TikTok Login

Uses TikTok's official OAuth 2.0 flow. Your password is never seen or stored by this app.

📥

Inbox Upload

Send a video to your TikTok inbox to review it privately before deciding to publish.

🚀

Direct Post

Publish a video immediately with configurable metadata such as title, privacy settings, and more.

How It Works

Launch the App

Open Short Video Uploader on your Windows or macOS desktop.

Log In with TikTok

Click "Login with TikTok" — a browser window opens and you authenticate directly on TikTok's official login page.

Select Your Video

Choose an MP4 file from your computer using the file picker.

Choose Upload Mode

Select "Inbox Upload" to privately review the video first, or "Direct Post" to publish immediately.

Upload

Click Upload — the app sends your video to TikTok via the official Content Posting API and shows you the result.

TikTok API Integration

Products & Permissions Used

This application integrates with the following TikTok for Developers products:

  • Login Kit — Allows users to securely authenticate with their TikTok account via OAuth 2.0. Uses the user.info.basic scope to identify the logged-in user.
  • Content Posting API — Enables uploading MP4 videos to TikTok on behalf of the authenticated user. Uses video.upload for inbox uploads and video.publish for direct posting.

The following OAuth scopes are requested:

user.info.basic video.upload video.publish

All API calls are made to TikTok's official endpoints. No user data is collected, stored, or shared by this application beyond what is necessary to perform the upload.
